# オシカド プライバシーポリシー

最終更新日: 2026年6月11日

## 基本方針

オシカド（以下「本アプリ」）は、ユーザーのプライバシーを最重要事項として設計されています。本アプリは個人情報を一切収集しません。

## 収集する情報

本アプリは以下のデータを収集しません：

- 氏名、メールアドレス等の個人識別情報
- 位置情報
- 写真・画像データ（端末外への送信は行いません）
- 使用状況・行動履歴
- 広告識別子

## データの保存場所

本アプリに入力されたすべてのデータ（推し名、イベント情報、写真等）はお使いの端末内にのみ保存されます。外部サーバーへの送信は一切行いません。

## 第三者への提供

収集するデータが存在しないため、第三者への提供は行いません。

## 広告

本アプリに広告は表示されません。
